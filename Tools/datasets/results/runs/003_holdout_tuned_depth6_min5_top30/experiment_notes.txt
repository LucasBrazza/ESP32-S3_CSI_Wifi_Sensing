Experiment 003 - Tuned holdout validation

Description:
Full classifier validation using the best decision tree configuration found in experiment 002.

Model:
Decision tree

Dataset:
Selected feature dataset with 790 samples and 30 selected features.

Parameters:
- Validation method: stratified holdout 80/20
- Random seed: 42
- Max tree depth: 6
- Min samples split: 5

Results:
- Accuracy: 0.620253
- Macro F1-score: 0.608970
- Weighted F1-score: 0.622193

Comparison with baseline experiment 001:
- Baseline accuracy: 0.582278
- Tuned accuracy: 0.620253
- Baseline macro F1-score: 0.534149
- Tuned macro F1-score: 0.608970
- Baseline weighted F1-score: 0.556645
- Tuned weighted F1-score: 0.622193

Main observation:
The tuned configuration improved the general performance of the classifier when compared with the baseline. The main improvement was related to a better balance between classes, especially because the baseline had poor recall for the empty class.

Embedded deployment observation:
Although the tuned tree is more complex than the baseline, the selected max depth of 6 is still compatible with future embedded inference, since each prediction requires only a small number of decision comparisons.

Next step:
Compare experiments 001 and 003 in a dedicated summary, then evaluate whether a stricter validation by source file is necessary.
