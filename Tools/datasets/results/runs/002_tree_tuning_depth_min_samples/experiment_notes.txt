Experiment 002 - Decision tree tuning

Description:
Controlled tuning experiment for the decision tree classifier, varying max_depth and min_samples_split.

Validation:
- Stratified holdout 80/20
- Random seed: 42

Dataset:
- Total samples: 790
- Train samples: 632
- Test samples: 158
- Features per sample: 30

Class distribution:
Class                    Total     Train      Test
empty                      197       158        39
static_presence            293       234        59
movement                   300       240        60

Tested values:
- max_depth: [3, 4, 5, 6, 8]
- min_samples_split: [2, 5, 10, 20]

Best result:
- max_depth: 6
- min_samples_split: 5
- accuracy: 0.620253
- macro F1-score: 0.608970
- weighted F1-score: 0.622193
- empty recall: 0.564103
- empty F1-score: 0.505747

Main observation:
This experiment compares different tree sizes while preserving the same dataset and validation split used in the baseline. The goal is to improve the balance between classification performance and model complexity for future embedded inference.
