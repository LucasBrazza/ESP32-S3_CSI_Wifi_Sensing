Experiment 001 - Holdout baseline

Description:
Initial decision tree validation using stratified holdout 80/20.

Model:
Decision tree

Dataset:
Selected feature dataset with 790 samples and 30 selected features.

Parameters:
- Validation method: stratified holdout 80/20
- Random seed: 42
- Max tree depth: 4
- Min samples split: 2

Results:
- Accuracy: 0.582278
- Macro F1-score: 0.534149
- Weighted F1-score: 0.556645

Main observation:
The classifier showed moderate initial performance. The main issue was the low recall for the empty class, which was mostly confused with static_presence. The results also varied considerably by quadrant, suggesting spatial dependency in the CSI response.

Next step:
Run controlled tuning experiments varying max_depth and min_samples_split, while preserving this experiment as the baseline.
